

Для програми виводу отриманої температури:
Компіляція:
```
g++ -o Start ./*.cpp -lsfml-graphics -lsfml-window -lsfml-system
```
Старт:
```
./Start
```

Для програми зчитування та передачі температури:

Компіляція:
```
g++ -o start ./*.cpp
```
Стари:
```
./Start
```
